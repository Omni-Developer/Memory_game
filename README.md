# 🧠 Memory Matching Game

A fun memory game built with HTML, CSS, and JavaScript using emojis! Try to match all the pairs as quickly and efficiently as possible.

## 🚀 Features
- Emoji-based cards
- Timer and move counter
- Responsive layout
- Reset button
- Victory message
